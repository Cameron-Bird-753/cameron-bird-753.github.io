Just Completed

Home Audio

I have a not so 'smart' Monoprice multizone amplifier, which is controlled with keypads. Luckily, it features a RS232 serial port which I used as an interface.

The frontend web application is built in Angular with Node.js used for the backend server and a SQLite database. I also decided to build out an Android mobile application using the Ionic framework.

The server is hosted on Rasberry PI, and connected to the amp. As Android only allows for HTTPS connections, the local server also runs a HTTPS protocol and involved a deep dive into HTTPS, SSL and TLS.
